# 🎓 School Assistant Bot

A simple AI-powered chatbot built using Python and Tkinter. This application helps students by answering questions clearly and simply, just like a teacher.

---

## Preview

![Demo](https://github.com/eshwarg14/AI-ML/raw/1d071baccd9a453fd7f93e2f198874aa548784fd/Images/SB.png)

---

## ✨ Features

- 🤖 AI-powered chatbot using OpenRouter API  
- 🎓 Designed as a school assistant  
- 💬 Clean chat interface using Tkinter  
- ⚡ Fast responses with threading  
- 🧠 Simple and easy-to-understand answers  
- 🔄 Real-time interaction  

---

## 🧠 Technologies Used

```
tkinter
requests
threading
```


## ▶️ Usage

```bash
python main.py
```

---

## 💬 How It Works

- 📡 Sends user input to OpenRouter API  
- 🤖 AI generates response using system prompt  
- 🧠 Response is displayed in chat UI  
- ⚡ Runs in background thread for smooth experience  

---

## 🔑 How to Get OpenRouter API Key

Follow these steps to create your API key:

1. 🌐 Go to OpenRouter:
   https://openrouter.ai

2. 🔐 Sign up / Login with your account

3. 👤 Click on your profile (top right corner)

4. 🔑 Go to **"API Keys"** section

5. ➕ Click **"Create Key"**

6. 📋 Copy your API key

---

## ⚠️ Important Security Tip

🚨 Never share your API key publicly (like in GitHub)

Replace this in your code with your original api key:

```python
API_KEY = ""
```
