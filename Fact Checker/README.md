# 🔍 AI Fake News Detector

A GUI-based AI application that analyzes news or text to determine whether it is likely **fake or real**. Built using Python, Tkinter, and OpenRouter API, this tool performs fact-checking using AI models.

## ✨ Features

- 🔍 Detects fake or misleading news  
- 🤖 AI-powered fact checking  
- 📝 Analyze any text input  
- ⚡ Fast response using API  
- 🖥️ Clean GUI using Tkinter  
- 📊 Displays detailed explanation  

---

## 🧠 What is Fake News Detection?

Fake news detection is the process of identifying whether a piece of information is:

- ❌ False or misleading  
- ✅ True or reliable  

👉 It helps in:
- Preventing misinformation  
- Improving awareness  
- Verifying news sources  

---

## 🤖 How AI Helps?

AI models:
- Understand text meaning  
- Compare facts with known knowledge  
- Provide reasoning-based answers

## 🔑 API Setup

1. 🌐 Go to: https://openrouter.ai  
2. 🔐 Login / Sign up  
3. 🔑 Generate API key  
4. Replace in code:

```python
self.api_key = "your-api-key"
```

---


## 🛠️ How It Works

- 📝 User enters news or text  
- 📡 Sends request to OpenRouter API  
- 🤖 AI analyzes the content  
- 📊 Returns fact-check result  
- 🖥️ Displays result in GUI
