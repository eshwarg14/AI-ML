# AI & Machine Learning
A collection of AI and machine learning projects built with Python - includes weather prediction, health monitoring, crop disease detection and traffic sign recognition

## Notes
This project demonstrates practical applications of AI and ML concepts.

## 📁 Repository Structure

```
AI-ML-Projects/
└── project/
   ├── main.py
   ├── requirements.txt
   └── README.md
```

> Each project has its own folder with its own README and requirements file.
> Dataset files are either included in the dataset/ folder or linked in the project README.

---

## 🛠️ Tech Stack

- Python 3.x
- scikit-learn
- Pandas / NumPy
- Matplotlib / Seaborn
- TensorFlow / Keras
- Streamlit (for UI-based projects)

---

## 🚀 Getting Started

1. Navigate to any project folder
   cd weather-predictor

2. Install dependencies
   pip install -r requirements.txt

3. Run the project
   python main.py

---

## 📌 Requirements

- Python 3.8 or above
- Dataset files (included in each project's dataset/ folder)
